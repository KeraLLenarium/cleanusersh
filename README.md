# cleanusersh
Skript for clean home directories on user\

Install
```
cd /setting
wget https://raw.githubusercontent.com/KeraLLenarium/cleanusersh/refs/heads/main/deldir.sh; chmod +x deldir.sh
```
Add $PATH
```
cp deldir.sh /usr/local/bin/clus; chmod +x clus
```
Use:
```
clus
```
Юзер: user
Успешно очищен
